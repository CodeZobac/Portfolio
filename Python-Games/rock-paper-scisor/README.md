Rock, Paper, Scissors Game

Overview:

    This Python script is a simple, text-based version of the classic game "Rock, Paper, Scissors". It features ASCII art for each choice and includes logic for playing against the computer.

How to Play:

    -Run the script.
    -When prompted, enter your choice: 'rock', 'paper', or 'scissors'.
    -The computer will randomly select its choice.
    -The script will display both choices and declare the winner.

Rules:

    -Rock crushes scissors.
    -Scissors cuts paper.
    -Paper covers rock.

Requirements:

    -Python 3.x

Running the Game:

    -To run the game, execute the script in a Python environment

Enjoy the game!
